System Architecture Notes

  • NAS: Primary storage device mounted at /media/scott/NAS/fileserver with exfat filesystem
  • Docker Services: Containerized services managed via docker-compose
  • Agent System: Hermes Agent core with skill management and automation capabilities
  • Integration: Services integrate through bind mounts, APIs, and shared storage
  • Security: Firewall rules, Docker security options, and Tailscale network isolation
